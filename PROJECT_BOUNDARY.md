# Project Boundary

`Kan-Recomentation` is an independent local project.

- it has its own `.git`
- it has no configured git remote by default
- it should be treated as its own workspace, not as an alternate checkout of another project
- earlier pre-split git metadata was backed up at `/tmp/kan-recomentation-git-backup-20260318-1`

Reference other projects only as external examples for API wiring, schema shape, or UI patterns.
Do not assume shared history, shared ownership, or interchangeable source files.
